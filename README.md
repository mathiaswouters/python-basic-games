# python-basic-games

### Set up a Python Virtual Environment
#### MacOS or Linux

1. `python3 -m venv .venv`
2. `source .venv/bin/activate`
3. `pip install -r requirements.txt`
        
#### Windows - recommendation to use git-bash terminal

1. `py -3 -m venv .venv`
2. `source .venv/Scripts/activate`
3. `pip install -r requirements.txt`
